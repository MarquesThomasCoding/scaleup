## Concept
Scale Up est une application destinée aux amateurs d'escalade, leur permettant de suivre leurs performances et de se mesurer à d'autres grimpeurs. L'application propose un classement en temps réel basé sur les performances des utilisateurs, avec une interface simple et intuitive.
